JavaScript 101:
Self-Paced Workshop

A self-paced, beginner-level workshop designed to introduce students to core programming principles and the JavaScript programming language.

Information

Requirements

This class assumes that you have a strong understanding of HTML and CSS. No JavaScript knowledge is expected or required.

Objectives

In this class, we will cover:

  • Processing.js: Drawing with JavaScript!
  • Variables
  • Animation basics
  • Strings
  • Functions
  • Logic, if statments, and control flow
  • Loops
  • Arrays
  • Objects
  • Object-oriented design basics

By the end of this class, you should be comfortable writing JavaScript, including events, DOM manipulation, and animation. You will be familiar with JavaScript objects and understand functions, variables, and primitive data types.

Class Materials and Schedule

We will begin day one with brief lectures on basic programming principles, pair programming, and how to use Khan Academy.

After the lectures, you'll work independently until you reach an exercise. The course materials include videos and practice sessions. Please use your headphones for all videos.

When you reach an exercise, put up a sticky note with the exercise name or number on it. A Teaching Assistant will help you find a pair partner to work on the exercise with.

If you get stuck at any point, raise your hand and a Teaching Assistant will come to help you through the problem.
